# Baseball PvP

A really small baseball game project

### Pre-Installing

Install node & yarn

### Installing

Install packages

```
yarn make
```

build react.js code

```
yarn build
```

run server

```
yarn server
```
or to run at customize port
```
PORT=3000 yarn server
```
